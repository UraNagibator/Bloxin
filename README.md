Этот проект демонстрирует продвинутую работу с Git в Visual Studio: клонирование удалённого репозитория, создание новой ветки, добавление файла, перенос коммита из другой ветки с помощью `git cherry-pick`, разрешение возможных конфликтов и отправка изменений на сервер.

## О проекте

Исходный репозиторий: `https://github.com/user/enhance-repo.git` (пример)

Цель задания:
- Клонировать репозиторий.
- Создать ветку `enhance` от основной ветки.
- Добавить файл `Feature.cs` с содержимым `Console.WriteLine("New feature");` и закоммитить его.
- Взять коммит из ветки `feature-old`, который изменяет файл `Feature.cs`, и применить его в текущей ветке через `cherry-pick`.
- При возникновении конфликтов разрешить их.
- Отправить готовую ветку `enhance` в удалённый репозиторий.
